A custom web browser developed using Java's Swing Library. 
It has multiple features like, maintaining browse history, bookmarks, can go back and forth from pervious visited pages, 
assign a home page, custom proxy to disable any website and a cool design. 

To Run This Project:
-Install Java JDK 8 on your system
-Create a java project in netbeans or elsewhere
-Copy all these files in that project (Copy .java files in src folder)
-Run Driver
